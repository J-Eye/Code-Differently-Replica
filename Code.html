Our Programs

1000 Kids Coding

The Code Differently 1000 Kids Coding program was created to
expose New Castle County students to computing and
programming. The 1000 Kids Coding courses are designed for all
experience levels, no experience required.

Pipeline DevShops

Pipeline DevShop is a youth work-based learning program. Youth
participants experience working in a real software development
environment while sharpening their technology and soft skills.
